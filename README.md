# js-dev-env
My Javascript Development Environment

# Requirements

* node >= 4.x
* npm >= 2.15

# Features

* **Package Manager**: npm
* **Development Web Server**: Localtunnel
* **Automation**: Express
* **Transpiler**: Babel
* **Bundler**: Webpack
* **Linter**: ESLint

# How to

Install application

```
npm install
```

Run application

```
npm start -s
```
